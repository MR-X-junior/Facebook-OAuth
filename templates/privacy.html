Kebijakan Privasi

Facebook OAuth - Privacy Policy

Terakhir diperbarui: 31 Oktober 2025

1. Pengantar

Selamat datang di Facebook OAuth. Kami sangat menghormati privasi Anda dan berkomitmen untuk melindungi data pribadi Anda. Kebijakan privasi ini menjelaskan bagaimana kami menangani informasi Anda ketika menggunakan layanan kami.

2. Informasi yang Kami Kumpulkan

Ketika Anda menggunakan layanan kami, kami dapat mengakses informasi berikut melalui Facebook API:

  • Informasi Profil: Nama, User ID, email, dan foto profil
  • Access Token: Token akses Facebook untuk autentikasi
  • Permissions: Izin yang Anda berikan kepada aplikasi
  • Pages Data: Informasi halaman Facebook yang Anda kelola (jika diizinkan)

3. Bagaimana Kami Menyimpan Data Anda

Penyimpanan Lokal:

Semua data Anda disimpan secara LOKAL di browser Anda menggunakan localStorage. Kami TIDAK menyimpan informasi pribadi Anda di server kami.

Apa artinya ini?

  • Data hanya tersimpan di perangkat Anda
  • Tidak ada transmisi data ke server kami
  • Anda memiliki kontrol penuh atas data Anda
  • Menghapus data browser akan menghapus semua informasi

4. Penggunaan Data

Data yang kami akses digunakan untuk:

  • Menampilkan informasi akun Anda
  • Menyediakan access token untuk keperluan Anda
  • Mengelola multiple akun Facebook
  • Memvalidasi status token dan akun

5. Keamanan Data

Kami mengimplementasikan langkah-langkah keamanan untuk melindungi informasi Anda:

  • Menggunakan HTTPS untuk semua koneksi
  • Data disimpan menggunakan localStorage browser yang aman
  • Tidak ada penyimpanan data di server
  • Validasi token dilakukan secara client-side

Catatan Penting:

Meskipun kami menggunakan praktik keamanan terbaik, tidak ada metode transmisi atau penyimpanan elektronik yang 100% aman. Kami tidak dapat menjamin keamanan absolut.

6. Berbagi Data dengan Pihak Ketiga

Kami TIDAK membagikan, menjual, atau menyewakan data pribadi Anda kepada pihak ketiga manapun. Data Anda tetap berada di browser Anda.

7. Cookie dan Teknologi Pelacakan

Aplikasi ini menggunakan localStorage browser untuk menyimpan preferensi dan data akun Anda. Kami tidak menggunakan cookie pelacakan atau teknologi iklan.

8. Hak Anda

Anda memiliki hak untuk:

  • Mengakses data yang tersimpan di browser Anda
  • Menghapus semua data kapan saja dengan membersihkan localStorage browser
  • Mencabut akses aplikasi melalui pengaturan Facebook Anda
  • Menghentikan penggunaan layanan kapan saja

9. Perubahan Kebijakan Privasi

Kami dapat memperbarui kebijakan privasi ini dari waktu ke waktu. Perubahan akan diposting di halaman ini dengan tanggal "Terakhir diperbarui" yang baru.

10. Kontak

Jika Anda memiliki pertanyaan tentang kebijakan privasi ini, silakan hubungi kami melalui halaman kontak.
