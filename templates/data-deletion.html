Instruksi Penghapusan Data

Cara menghapus data Anda dari aplikasi kami

Informasi Penting
Privasi Anda Adalah Prioritas: Aplikasi ini TIDAK menyimpan data pribadi Anda di server kami. Semua data disimpan secara lokal di localStorage browser Anda.
Data Yang Kami Kumpulkan

Aplikasi kami menyimpan informasi berikut secara lokal di browser Anda:

Informasi Pribadi
  • Nama Facebook
  • User ID Facebook
  • Alamat Email (jika diberikan)
  • URL Foto Profil
Token Akses
  • Facebook Access Token
  • Tipe Token dan Tanggal Kadaluarsa
  • Token Akses Page (jika ada)
Data Permissions
  • Daftar permissions yang diberikan
  • Informasi Facebook Pages
  • Status permission (granted/declined)
Catatan: Data ini disimpan HANYA di localStorage browser Anda dan TIDAK dikirim atau disimpan di server kami.
Cara Menghapus Data Anda

Metode 1: Hapus Melalui Aplikasi

Cara termudah untuk menghapus data Anda adalah langsung dari aplikasi kami:

1
Buka Halaman Utama

Navigasi ke halaman utama dimana semua akun Anda terdaftar

2
Klik Pada Akun Anda

Klik pada card akun untuk melihat detail akun

3
Hapus Akun

Scroll ke bawah dan klik tombol merah "Hapus Akun"

4
Konfirmasi Penghapusan

Konfirmasi pilihan Anda pada dialog popup yang muncul

Metode 2: Hapus Data Browser

Anda juga dapat menghapus semua data dengan membersihkan localStorage browser:

Google Chrome
  1. Tekan F12 untuk membuka Developer Tools
  2. Buka tab Application
  3. Di sidebar kiri, expand Local Storage
  4. Klik pada nama domain
  5. Cari dan hapus key fb_accounts
  6. Atau klik Clear All untuk menghapus semua data
Mozilla Firefox
  1. Tekan F12 untuk membuka Developer Tools
  2. Buka tab Storage
  3. Expand Local Storage
  4. Klik pada nama domain
  5. Klik kanan pada fb_accounts dan pilih Delete
Safari
  1. Aktifkan menu Developer: Preferences → Advanced → centang "Show Develop menu"
  2. Buka Develop → Show Web Inspector
  3. Klik tab Storage
  4. Pilih Local Storage
  5. Hapus entry fb_accounts
Microsoft Edge
  1. Tekan F12 untuk membuka Developer Tools
  2. Buka tab Application
  3. Expand Local Storage di sidebar kiri
  4. Klik pada domain
  5. Hapus fb_accounts atau klik Clear All

Pertanyaan Umum (FAQ)
Apakah data saya benar-benar aman?
Ya, sangat aman. Data Anda hanya tersimpan di localStorage browser Anda sendiri dan tidak pernah dikirim ke server kami. Kami menerapkan HTTPS untuk semua koneksi dan berbagai security measures lainnya.
Berapa lama proses penghapusan data?
Penghapusan data bersifat instant. Begitu Anda menghapus akun dari aplikasi atau membersihkan localStorage browser, data langsung terhapus seketika.
Apakah saya bisa menggunakan aplikasi lagi setelah menghapus data?
Ya, tentu saja. Anda dapat login kembali kapan saja dan data baru akan tersimpan. Penghapusan data tidak memblokir akses Anda ke aplikasi.
Apakah Facebook masih memiliki data saya setelah penghapusan?
Data yang tersimpan di Facebook dikontrol oleh Facebook, bukan oleh aplikasi kami. Untuk menghapus data dari Facebook, Anda perlu melakukannya melalui pengaturan Facebook atau mencabut akses aplikasi.
Bagaimana jika saya menggunakan aplikasi di beberapa browser?
Data disimpan per browser. Jika Anda menggunakan aplikasi di Chrome dan Firefox, data tersimpan terpisah. Anda perlu menghapus data di masing-masing browser secara terpisah.
Apakah token saya masih valid setelah penghapusan dari aplikasi?
Token tetap valid di sisi Facebook sampai expired atau Anda mencabut akses aplikasi. Namun, token yang dihapus dari aplikasi kami tidak dapat digunakan lagi dalam aplikasi ini kecuali Anda login ulang.
