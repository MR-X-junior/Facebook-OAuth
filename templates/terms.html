2. Deskripsi Layanan

Facebook OAuth adalah alat bantu yang memungkinkan Anda untuk:

  • Mendapatkan access token Facebook melalui OAuth
  • Mengelola multiple akun Facebook
  • Menyimpan informasi akun secara lokal di browser
  • Memvalidasi status dan permissions token

Catatan: Layanan ini bersifat gratis dan disediakan "sebagaimana adanya" tanpa jaminan apapun.

5. Keamanan Access Token

Access token adalah kredensial sensitif yang memberikan akses ke akun Facebook Anda:

  • Jangan bagikan access token Anda kepada siapapun

  • Hapus token jika tidak lagi digunakan
  • Ganti password Facebook jika token bocor
  • Cabut akses aplikasi melalui pengaturan Facebook jika curiga

Anda bertanggung jawab penuh atas keamanan access token Anda. Developer tidak bertanggung jawab atas penyalahgunaan token yang terjadi karena kelalaian Anda.
